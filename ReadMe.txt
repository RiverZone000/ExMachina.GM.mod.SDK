SDK для игры Ex Machina

Комплектация:

.vscode - плагин от ThePlain для Visual Studio Code для более удобного редактирования lua-скриптов и триггеров.
Docs - различная документация к редактору карт, созданию квестов и прочие полезности.
M3DEditor - редактор карт SDK.
SciTE - старая версия текстового редактора SciTE, которым пользовались разработчики.
ConversationEditor.exe - редактор диалогов и квестов.
TME rus.exe; TME eng.exe - программа для просмотра и настройки игровых моделей.

Установка:

1. Скопировать содержимое архива в папку с установленной игрой, согласиться на замену необходимых файлов.
2. Перейти в папку M3DEditor, создать ярлык файла M3DEditor.exe (клик правой кнопкой мыши --> Создать ярлык)
3. Зайти в свойства ярлыка (клик правой кнопкой мыши НА ЯРЛЫКЕ --> Свойства), во вкладке "Ярлык" в строке "Рабочая папка" стереть в конце "M3DEditor".
4. Скопировать ярлык в папку с установленной игрой (куда вы до этого распаковали архив) и запускать редактор карт ТОЛЬКО с помощью него.
